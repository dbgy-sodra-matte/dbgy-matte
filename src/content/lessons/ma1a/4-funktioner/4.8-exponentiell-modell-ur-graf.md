## Hitta funktionen från en graf

Det här är baklängesriktningen: i stället för att räkna ur en funktion ska du **läsa ut funktionen** ur en graf. Du letar efter **C** och **a** och skriver `y = C · a^x`.

## Steg 1: hitta C (startvärdet)

C är y-värdet där kurvan skär y-axeln, alltså vid **x = 0**.

> Kurvan går genom (0, 300) → C = 300.

## Steg 2: hitta a (förändringsfaktorn)

Dela y-värdet vid x = 1 med y-värdet vid x = 0:

```
a = y(1) / y(0)
```

> (0, 300) och (1, 600): a = 600 / 300 = **2**.

## Steg 3: skriv ihop

> C = 300, a = 2 → **y = 300 · 2^x**

## Halveringstid

När något **halveras** med jämna mellanrum är förändringsfaktorn **0,5** per period. Räkna ut hur många perioder det är, och använd `0,5` upphöjt till antalet:

> 80 mg, halveras varannan timme. Efter 4 timmar = 2 halveringar:
> `80 · 0,5² = 80 · 0,25 = 20 mg`.

## Gråsälar — först linjärt, sedan exponentiellt

En klassisk A-uppgift: något växer först **linjärt** (samma antal per år) och **sedan exponentiellt** (samma faktor per år). Grafen är först en rak linje som sedan böjer uppåt. Räkna varje fas för sig — linjär med tillägg, exponentiell med faktor.

> **Regel att skriva ner:** C = y vid x = 0. a = y(1) / y(0). Skriv y = C · a^x. Halvering → faktor 0,5 per period.
