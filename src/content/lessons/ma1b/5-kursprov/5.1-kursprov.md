worked_example: |
  ## Inför kursprovet — checklista över de viktigaste teknikerna

  ### Algebra
  - Förenkla uttryck (slå ihop liknande termer, multiplicera in i parentes, hantera minustecken)
  - Utveckla (x + a)(x + b) — fyra delprodukter, slå ihop mellantermerna
  - Lös linjära ekvationer (ax + b = c, x i båda leden, parenteser, nämnare)

  ### Ekonomi
  - Förändringsfaktor (+x % → 1+x/100; −x % → 1−x/100)
  - Upprepad förändring: slutvärde = startvärde · (faktor)^n
  - Räkna baklänges: dela med faktorn

  ### Funktioner
  - Linjär: y = kx + m (k = lutning, m = skärning med y-axeln)
  - Exponential: y = C · a^x (a > 1 = tillväxt, a < 1 = avtagande)
  - Potens: y = C · x^a (x i basen, NOT i exponenten)

  ### Fördjupning
  - Nollproduktregeln: a · b = 0 → a = 0 eller b = 0
  - Konjugatregeln: a² − b² = (a − b)(a + b)
  - Faktorisering av x² + bx + c: hitta talen p, q där p·q = c och p + q = b

  ## Sju vanligaste fällorna att undvika

  | Fälla | Hur du undgår den |
  |---|---|
  | Glömma minustecken framför parentes | Skriv ALLA tecken som vänts |
  | Glömma negativa lösningen vid x² = a | x = ±√a, inte bara +√a |
  | Addera procenttal istället för att multiplicera faktorer | Multiplicera ALLTID förändringsfaktorerna |
  | Tappa enheter (kr, m, år) | Skriv ut enheten i slutsvaret |
  | Inte kontrollera svaret | Sätt in i originalekvationen |
  | Avrunda för tidigt | Räkna med exakta tal, avrunda sist |
  | Glömma tolka tillbaka | "X = 8" betyder vad i situationen? |

  ## Tidsstrategi

  1. **Läs ALLA uppgifter** först — välj en lätt att starta med
  2. **Sätt timmar** per uppgift — fastna inte
  3. **Lämna inget tomt** — skriv minst en formel även om du inte kan slutföra
  4. **Spara 10 min på slutet** för att granska tecken och beräkningar

      hint2: "Olika tecken på p och q (negativ produkt). Pröva (4, −3): 4·(−3) = −12, 4 + (−3) = 1 ✓. Faktorisera och använd nollproduktregeln."

      solution: |
        V(t) = 320 000 · 0,88^t

        t = 9: V ≈ 102 200 (över 100 000)
        t = 10: V ≈ 89 950 (under!)

        2x + 2(x + 4) = 28

        **x = 5 m** (kort sida)

## Inför kursprovet

Det här är slutprovet för hela Ma1b. Det testar **allt** vi gått igenom under terminen.

## Provets struktur (vanligen)

Kursprovet i Ma1b består typiskt av två delar:

**Del 1 — utan miniräknare/hjälpmedel**
- Korta uppgifter, ofta E-nivå
- Snabba räkningar, grundförståelse
- Du visar att du kan grunderna

**Del 2 — med miniräknare/hjälpmedel**
- Längre uppgifter, högre nivå (C och A)
- Tillämpningar, modellering, problemlösning
- Här ska du visa förståelse och resonemang

(Exakt format kan variera — fråga din lärare!)

## Tips inför kursprovet

### Studeteknik
1. **Gör om gamla uppgifter** — från lektionerna, duggorna och proven.
2. **Skapa egna sammanfattningar** av varje moment.
3. **Identifiera vad som är svårast** för dig och lägg mer tid där.
4. **Räkna gamla NP** om läraren har dem tillgängliga.

### Under provet
1. **Läs ALLA uppgifter först.** Börja med dem du tycker är lättast.
2. **Skriv tydliga lösningar.** Inte bara svar — visa stegen.
3. **Kontrollera dina svar.** Sätt in i originalekvationen.
4. **Tolka svaren i situationen.** "Det betyder X kr/månad..."
5. **Tid:** sätt ungefärlig tid per uppgift. Fastna inte på en svår.

### Vanliga fällor att undvika
- Glömma minustecken framför parentes
- Glömma att kvadratrots-ekvationer har två lösningar (±)
- Bara addera procent vid upprepad förändring
- Tappa enheter (kr, m, år)
- Inte tolka svaret tillbaka i situationen

## Mixad övning — pre-test

Övningarna nedan blandar moment och nivåer. Använd dem som en testkörning inför kursprovet — om du klarar dem utan att tveka är du redo.

## Och slutligen

**Lycka till!** Du har gjort en hel kurs. Du kan mycket mer än du tror.
