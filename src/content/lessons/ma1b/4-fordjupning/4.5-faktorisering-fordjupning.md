  | Uttryck | Faktoriserat |
  |---|---|
  | x² − 16 | (x − 4)(x + 4) |

  Sök p och q så att:
  - **p · q = c** (sista konstanten)

  Då är faktoriseringen `(x + p)(x + q)`.

  | Uttryck | p, q | Faktoriserat |
  |---|---|---|
  | x² + 7x + 12 | 3, 4 | (x + 3)(x + 4) |
  | x² − 5x + 6 | −2, −3 | (x − 2)(x − 3) |
  | x² + 2x − 8 | 4, −2 | (x + 4)(x − 2) |

  ### Tecken-tabell för p och q

  | c | b | p och q |
  |---|---|---|
  | + | + | båda positiva |
  | + | − | båda negativa |
  | − | + | olika tecken, positivt vinner |
  | − | − | olika tecken, negativt vinner |

  Faktorisera → använd nollproduktregeln på varje faktor.

  ```
  x² + 3x − 10 = 0
  (x + 5)(x − 2) = 0
  x = −5 eller x = 2
  ```

## Vad är faktorisering?

**Faktorisering** är att skriva om en summa som en **produkt**.

Det är motsatsen till att multiplicera in i en parentes:

- **Multiplicera in:** 3(x + 2) → 3x + 6
- **Faktorisera:** 3x + 6 → 3(x + 2)

Varför vill vi faktorisera? Tre huvudskäl:

1. **Lösa ekvationer** via nollproduktregeln
2. **Förenkla bråk** (gemensamma faktorer kan strykas)
3. **Förstå strukturen** — vad uttrycket "består av"

## Tre tekniker för faktorisering

### 1. Bryta ut gemensam faktor

Det enklaste fallet. Hitta något som alla termer har gemensamt.

```
4x + 12 = 4(x + 3)        ← 4 är gemensamt
6x² − 9x = 3x(2x − 3)     ← 3x är gemensamt
```

**Trick:** kolla först om det finns ett **gemensamt tal**. Sedan en **gemensam variabel**.

### 2. Konjugatregeln

```
a² − b² = (a − b)(a + b)
```

Det här mönstret kallas **konjugatregeln**. Det fungerar när du har en differens av två kvadrater.

```
x² − 16 = (x − 4)(x + 4)
x² − 49 = (x − 7)(x + 7)
4x² − 9 = (2x − 3)(2x + 3)
```

### 3. Hitta två tal (för andragradsuttryck)

För `x² + bx + c` letar du två tal **p** och **q** så att:

```
p · q = c     (produkten är c)
p + q = b     (summan är b)
```

Då är faktoriseringen `(x + p)(x + q)`.

```
x² + 7x + 12     →   3 · 4 = 12, 3 + 4 = 7   →   (x + 3)(x + 4)
x² − 5x + 6      →   (−2)·(−3) = 6, −2−3 = −5  →  (x − 2)(x − 3)
x² + 2x − 8      →   4 · (−2) = −8, 4 + (−2) = 2  →  (x + 4)(x − 2)
```

**Tecken-tabell** för att veta vilka tecken p och q ska ha:

| c | b | p och q |
|---|---|---|
| + | + | Båda positiva |
| + | − | Båda negativa |
| − | vilket som | Ett positivt, ett negativt |

## Worked example — lös ekvation genom faktorisering

> *Lös: x² + 5x = 0*

**Steg 1:** Faktorisera. Bryt ut x.
```
x² + 5x = x(x + 5)
```

**Steg 2:** Sätt produkten = 0 och använd nollproduktregeln.
```
x(x + 5) = 0
x = 0  eller  x + 5 = 0 → x = −5
```

**Lösningar: x = 0 eller x = −5.**

## Worked example — konjugatregeln

> *Lös: x² − 81 = 0*

```
x² − 81 = (x − 9)(x + 9) = 0
x = 9  eller  x = −9
```

## Worked example — hitta två tal

> *Lös: x² − 6x + 8 = 0*

Sök p, q: produkt 8, summa −6.

Båda negativa (eftersom produkt är + och summa är −):
- (−1) och (−8): produkt 8, summa −9 ❌
- (−2) och (−4): produkt 8, summa −6 ✓

```
x² − 6x + 8 = (x − 2)(x − 4) = 0
x = 2  eller  x = 4
```

## När fungerar inte de tre metoderna?

Vissa andragradsuttryck kan inte faktoriseras med heltal — t.ex. `x² + x − 1`. Då behövs en mer avancerad metod (kvadratiska formeln, som vi rör vid i Ma2).

Men i Ma1b räcker de tre metoderna för i princip alla uppgifter.

## Tips

- **Bryt ut FÖRST.** Om det finns gemensam faktor är det enklaste steget.
- **Sedan kolla konjugatregeln.** Är det en differens av två kvadrater?
- **Sist hitta två tal-metoden.** För x² + bx + c.
- **Kontrollera genom att multiplicera in.** Du ska få tillbaka originalet.
