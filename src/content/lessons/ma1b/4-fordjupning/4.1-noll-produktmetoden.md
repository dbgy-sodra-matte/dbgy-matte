worked_example: |
  **Nollproduktregeln:** om `a · b = 0`, så är minst en av a och b lika med 0.

  Det är inte möjligt att gångar ihop två tal (där båda är skilda från noll) och få noll. Det är grunden.

  ## 1. Två faktorer — `(x − 3)(x + 7) = 0`

  Antingen är (x − 3) = 0 eller (x + 7) = 0:

  - x − 3 = 0 → **x = 3**
  - x + 7 = 0 → **x = −7**

  **Två lösningar:** x = 3 eller x = −7.

  ## 2. Bryt ut först — `x² + 5x = 0`

  Bryt ut x:
  ```
  x² + 5x = x(x + 5) = 0
  ```

  Nu nollproduktregeln:
  - x = 0
  - eller x + 5 = 0 → x = −5

  ## 3. Konjugatregeln — `x² − 9 = 0`

  Faktorisera först:
  ```
  x² − 9 = (x − 3)(x + 3) = 0
  ```

  Sedan: x = 3 eller x = −3.

  ## 4. Andragradsekvation — `x² − 7x + 12 = 0`

  Sök två tal som ger:
  - Produkt: **12** (sista konstanten)
  - Summa: **7** (talet framför x, med tecken vänt)

  Talen 3 och 4 fungerar (3·4 = 12, 3+4 = 7). Faktorisera:
  ```
  (x − 3)(x − 4) = 0
  ```

  Lösningar: x = 3 eller x = 4.

  > **Trick:** nollproduktregeln fungerar bara när högerledet är **0**. Allt annat måste flyttas över först.

## Nollproduktregeln — det är hela hemligheten

Säg du har ekvationen:

```
a · b = 0
```

Vad kan vi säga om a och b?

**Minst en av dem måste vara noll.** Det är inte möjligt att gånga ihop två tal (där båda är skilda från noll) och få noll.

Det här är **nollproduktregeln**, och den är hela basen för metoden.

## Varför är detta användbart?

För att vi kan använda det för att **lösa kvadratiska ekvationer** — om vi får dem på rätt form.

> *Lös: x(x − 5) = 0*

Vi har två faktorer: x och (x − 5). Produkten är 0.

→ x = 0 eller x − 5 = 0 (alltså x = 5).

**Lösningar: x = 0 eller x = 5.**

Tänk dig hur svårt det skulle vara att lösa **utan** nollproduktregeln. Med den blir det två enkla linjära ekvationer.

## Worked example — två parenteser

> *Lös: (x − 3)(x + 7) = 0*

Två faktorer: (x − 3) och (x + 7). Produkten är 0.

- x − 3 = 0 → x = 3
- x + 7 = 0 → x = −7

**Lösningar: x = 3 eller x = −7.**

## Worked example — bryt ut

Ibland står ekvationen inte direkt i faktoriserad form. Då behöver vi **faktorisera** först.

> *Lös: x² + 5x = 0*

Bryt ut gemensam faktor (x):

```
x² + 5x = x · x + 5 · x = x · (x + 5)
```

Då blir det:

```
x(x + 5) = 0
```

Och med nollproduktregeln:

- x = 0
- x + 5 = 0 → x = −5

**Lösningar: x = 0 eller x = −5.**

## När fungerar inte nollproduktregeln?

**Den fungerar bara när produkten är 0.**

> *x(x − 3) = 4 — kan vi säga x = 4 eller x − 3 = 4? NEJ.*

Det här är ett vanligt fel. Nollproduktregeln gäller **bara** när produkten är 0. Annars måste man göra något annat — t.ex. flytta över allt till en sida först.

> *x(x − 3) = 4 → x² − 3x = 4 → x² − 3x − 4 = 0 → (x − 4)(x + 1) = 0 → x = 4 eller x = −1*

## Faktorisering — det andra konsten

För att kunna använda nollproduktregeln måste ekvationen vara på formen **produkt = 0**. Det betyder att vi ofta måste **faktorisera**.

**Tre vanliga sätt att faktorisera:**

**1. Bryt ut gemensam faktor:**
```
x² + 5x = x(x + 5)
3x² − 6x = 3x(x − 2)
```

**2. Konjugatregeln:** `a² − b² = (a − b)(a + b)`
```
x² − 9 = (x − 3)(x + 3)
x² − 25 = (x − 5)(x + 5)
```

**3. Hitta två tal som passar:** `x² + bx + c = (x + p)(x + q)` där p · q = c och p + q = b.
```
x² − 7x + 12 = (x − 3)(x − 4)
x² + 8x + 15 = (x + 3)(x + 5)
```

Detta sista trick är inte alltid uppenbart, men det fungerar för många "vanliga" exempel.

## Strategi-checklista

1. **Är ekvationen på formen `produkt = 0`?**
   - Ja → gå till steg 3
   - Nej → faktorisera först (steg 2)
2. **Faktorisera** — använd brytning, konjugatregeln, eller hitta två tal.
3. **Använd nollproduktregeln** — sätt varje faktor lika med 0 och lös.
4. **Kontrollera** genom insättning i originalekvationen.

## Tips

- **Allt på en sida först.** `x² = 4x` → `x² − 4x = 0` → faktorisera.
- **Glöm inte trivlösningar.** Om x är en av faktorerna är x = 0 ofta en lösning.
- **Två lösningar är normalt** för andragradsekvationer — men ibland sammanfaller de eller fattas.
