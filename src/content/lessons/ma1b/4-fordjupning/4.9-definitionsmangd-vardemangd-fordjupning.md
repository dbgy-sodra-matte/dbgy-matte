worked_example: |
  Tre principer för **definitionsmängd**:

  1. **Division med 0 förbjudet** — sök nollställen i nämnaren
  2. **Kvadratrot av negativt förbjudet** — inuti rot ≥ 0
  3. **I tillämpningar — realistiska gränser** — tid ≥ 0, antal är heltal, längd > 0

  ## Två slags begränsningar samtidigt

  **Funktion: y = √(x − 2) / (x − 5)**

  Två villkor:
  - Inuti roten: x − 2 ≥ 0 → x ≥ 2
  - Nämnaren ≠ 0: x ≠ 5

  Kombinerat: **x ≥ 2 och x ≠ 5.**

  ## Värdemängd för andragrad — y = x² + k

  Parabler har ett **minsta värde** (om de öppnar uppåt) eller **största värde** (om de öppnar nedåt).

  | Funktion | Minsta y | Värdemängd |
  |---|---|---|
  | y = x² | 0 | y ≥ 0 |
  | y = x² + 3 | 3 | y ≥ 3 |
  | y = (x − 1)² + 5 | 5 | y ≥ 5 |
  | y = −x² + 2 | största y = 2 | y ≤ 2 |

  ## Värdemängd för exponential

  | Funktion | Värdemängd |
  |---|---|
  | y = 2^x | y > 0 (alltid positivt) |
  | y = 2^x + 5 | y > 5 (skiftat upp) |

  Exponentialer kan **aldrig** nå 0 — de närmar sig men når aldrig.

  ## Komplex fall — funktion med "hål"

  ```
  y = (x² − 4x + 3) / (x − 1)
    = (x − 1)(x − 3) / (x − 1)       ← faktorisera täljaren
    = x − 3              (för x ≠ 1)
  ```

  Funktionen är en linje y = x − 3, MEN x = 1 är förbjudet. Det betyder att y = 1 − 3 = −2 aldrig kan inträffa.

  **Värdemängd: alla y utom −2.** Grafen har ett "hål" vid (1, −2).

## Repetition + djupare

I 3.12 lärde vi oss grunderna. Här utvidgar vi till funktioner med **flera typer av begränsningar**:

- Funktioner med rot och bråk samtidigt
- Värdemängd för andragradsfunktioner
- Funktioner med absolutbelopp eller komplexa uttryck

## Tre principer för definitionsmängd

1. **Division med 0 förbjudet.** Sök nollställen i nämnare.
2. **Kvadratrot av negativt förbjudet.** Inuti rot ska vara ≥ 0.
3. **I tillämpningar — realistiska gränser.** Tid ≥ 0, antal är heltal, etc.

## Värdemängd — andragrad

Andragradsfunktioner (parabler) har alltid ett **minsta värde** (om den öppnar uppåt) eller **största värde** (om den öppnar nedåt).

```
y = x² + 3      → minsta y = 3 (vid x = 0)
y = (x − 1)² + 5 → minsta y = 5 (vid x = 1)
y = −x² + 2     → STÖRSTA y = 2 (öppnar nedåt)
```

Värdemängden är allt från minsta upp till oändligheten (eller motsvarande omvänt).

## Värdemängd — exponential

```
y = 2^x          → y > 0 (alltid positivt, aldrig 0)
y = 3 · 2^x      → y > 0
y = −2^x         → y < 0
y = 2^x + 5      → y > 5 (skiftat uppåt)
```

Exponentialfunktioner är alltid positiva (eller alltid negativa, beroende på tecken).

## Tillämpning — verklig kontra matematisk definitionsmängd

| Funktion | Matematisk D | Verklig D |
|---|---|---|
| K(t) = 80 + 25t (cykeluthyrning) | alla reella | t ≥ 0 (rimligen ≤ 24) |
| B(t) = 100 · 2^t (bakterier) | alla reella | t ≥ 0 |
| V(t) = 500 000 · 0,85^t (bil) | alla reella | t ≥ 0, kanske ≤ 20 |
| A(s) = s² (kvadratens area) | alla reella | s > 0 |

## Strategi

1. **Identifiera begränsningar.** Bråk? Rot? Realistisk kontext?
2. **Skriv olikheten** för varje begränsning.
3. **Kombinera** om det finns flera.
4. **Värdemängd:** vad är minsta/största möjliga y? Kan funktionen ge alla värden mellan?
