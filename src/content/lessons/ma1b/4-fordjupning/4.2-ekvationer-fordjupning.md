worked_example: |
  En linjär ekvation kan ha tre utfall.

  ## 1. En unik lösning (det vanliga)

  ```
  2(x + 1) − 3 = x + 4
  2x + 2 − 3 = x + 4
  2x − 1 = x + 4
  x = 5
  ```

  ## 2. Ingen lösning

  Om ekvationen leder till ett falskt påstående:

  ```
  4x − 5 = 4x + 1
  −5 = 1      ← falskt!
  ```

  Inget x kan göra båda sidor lika. **Ingen lösning.**

  Geometriskt: två räta linjer med samma lutning men olika nivå (parallella).

  ## 3. Oändligt många lösningar

  Om ekvationen leder till `0 = 0`:

  ```
  3(x + 4) − 2x = x + 12
  3x + 12 − 2x = x + 12
  x + 12 = x + 12
  0 = 0 ✓
  ```

  Alla x uppfyller ekvationen — det är **samma linje** på båda sidor.

  ## Komplex linjär ekvation — flera parenteser

  ```
  4(x + 2) − 3(x − 1) = 14
  4x + 8 − 3x + 3 = 14     ← multiplicera in NOGA med tecknen
  x + 11 = 14
  x = 3
  ```

  > **Varning:** `−3(x − 1) = −3x + 3` (inte −3x − 3). Minustecknet vänder båda tecknen.

  ## Andragradsekvation med nollproduktregeln

  ```
  (x − 4)(x + 2) = 0
  x − 4 = 0  →  x = 4
  x + 2 = 0  →  x = −2
  ```

  ## Ekvation med nämnare — bli av med dem genom multiplikation

  ```
  x/3 + x/6 = 5        | · 6
  2x + x = 30
  3x = 30
  x = 10
  ```

## Tre möjliga utfall vid ekvationslösning

När du löser en linjär ekvation kan tre saker hända:

**1. En unik lösning** — det vanliga. T.ex. x = 5.

**2. Inga lösningar** — ekvationen leder till ett falskt påstående, t.ex. `5 = 7`.

**3. Oändligt många lösningar** — ekvationen leder till `0 = 0` eller `x = x`. Då fungerar alla x.

## Worked example — unik lösning

> *2(x + 1) − 3 = x + 4*

```
2x + 2 − 3 = x + 4
2x − 1 = x + 4
x = 5
```

**En lösning: x = 5.**

## Worked example — ingen lösning

> *4x − 5 = 4x + 1*

```
4x − 5 = 4x + 1
−5 = 1   ← falskt!
```

Inget x kan lösa detta. **Ingen lösning.**

Geometriskt: två räta linjer med samma lutning men olika nivå (parallella).

## Worked example — oändligt många lösningar

> *3(x + 4) − 2x = x + 12*

```
3x + 12 − 2x = x + 12
x + 12 = x + 12
0 = 0 ✓
```

Båda sidor är **alltid lika** — oavsett x. **Oändligt många lösningar.**

Geometriskt: två räta linjer som är identiska (samma lutning, samma nivå).

## Komplexa ekvationer — flera parenteser

> *4(x + 2) − 3(x − 1) = 14*

Steg-för-steg:

```
4(x + 2) − 3(x − 1) = 14
4x + 8 − 3x + 3 = 14     ← multiplicera in noga
x + 11 = 14
x = 3
```

**Varning:** `−3(x − 1) = −3x + 3`. Minustecknet vänder båda tecknen inuti.

## Ekvationer med nämnare

Få bort nämnarna genom att multiplicera med gemensam nämnare.

> *x/3 + x/6 = 5*

Gemensam nämnare: 6.

```
x/3 + x/6 = 5     | · 6
2x + x = 30
3x = 30
x = 10
```

## Andragradsekvationer — i faktoriserad form

Vissa är enkla att lösa direkt:

> *(x − 4)(x + 2) = 0 → x = 4 eller x = −2*

> *x² = 25 → x = ±5*

> *3x² − 12 = 0 → x² = 4 → x = ±2*

## Strategi-checklista

1. **Multiplicera in i alla parenteser** — håll noga koll på tecknen.
2. **Få bort nämnare** genom multiplikation.
3. **Samla x på ena sidan**, tal på andra.
4. **Dela med koefficienten** framför x.
5. **Kontrollera** lösningen.
6. **Var medveten om specialfall** — ingen eller oändliga lösningar.
