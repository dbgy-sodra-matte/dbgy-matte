## Vad är GeoGebra?

**GeoGebra** är ett gratis program (web, dator, mobil) för matematik. Det hjälper dig:

- **Rita grafer** av funktioner direkt
- **Hitta skärningar** och nollställen visuellt
- **Lösa ekvationer** med CAS-verktyget
- **Konstruera** geometri och se sammanhang

Du kommer åt det på **geogebra.org** — eller ladda ner appen.

## Grunderna — rita en funktion

I "input"-fältet (botten), skriv:
```
y = 2x + 3
```
eller använd funktionsnotation:
```
f(x) = 2x + 3
```

Grafen ritas direkt. Du kan ha flera funktioner — de får olika färger automatiskt.

## Hitta nollställen (där grafen skär x-axeln)

Tre sätt:

1. **Visuellt** — kolla där grafen skär x-axeln.
2. **Verktyget "Nollställen"** — markera funktionen och klicka.
3. **Kommando** — skriv `Nollställen(f)` i input.

> *f(x) = x² − 9 → nollställen vid (−3, 0) och (3, 0).*

## Hitta skärningen mellan två grafer

1. **Skriv båda funktionerna**: f(x) = ... och g(x) = ...
2. **Verktyget "Skärning"** eller skriv `Skärning(f, g)` i input.
3. GeoGebra markerar skärningspunkten/punkterna med koordinater.

## CAS-läge — lös ekvationer symboliskt

CAS = Computer Algebra System. I GeoGebra finns ett separat CAS-fönster där du kan:

- Lösa ekvationer: `Lös(x² − 5x + 6 = 0, x)` → x = 2, x = 3
- Förenkla uttryck
- Beräkna derivator (Ma2)

Bra för att kontrollera dina handräknade svar.

## Mall — så löser du ett problem grafiskt

> *Lös ekvationen x² + 2x − 3 = 0.*

**I GeoGebra:**
1. Skriv `f(x) = x² + 2x − 3` i input.
2. Använd "Nollställen"-verktyget på grafen.
3. GeoGebra markerar (−3, 0) och (1, 0).

→ **Lösningar: x = −3 och x = 1.**

(Du kan också göra det med papper och penna — faktorisera till (x + 3)(x − 1) = 0.)

## När GeoGebra är bättre än för hand

- **Komplicerade ekvationer** som inte går att lösa exakt
- **Många funktioner att jämföra** visuellt
- **Visualisera** vad en parameter gör (skifta linjer, ändra k och m)
- **Kontrollera** ett handräknat svar
- **Skapa snygga grafer** för redovisning

## När du bör räkna för hand istället

- **Enkla ekvationer** — snabbare i huvudet
- **Lära dig metoderna** — du måste kunna det vid prov
- **Bedömningssituationer** där hjälpmedel inte är tillåtna

## Tips

- **Lär dig kommandona.** `Skärning`, `Nollställen`, `Lös` är de vanligaste.
- **Använd för kontroll.** Räkna för hand först, kolla med GeoGebra.
- **Animera vid behov** — med skjutreglage kan du se hur k påverkar y = kx + m direkt.
- **Spara filerna** för senare användning.

## Övningar i denna lektion

Övningarna nedan har lösningar som du kan räkna för hand. **Använd GeoGebra för att verifiera dina svar** efter att du löst dem analytiskt — det är så du lär dig båda färdigheter.
