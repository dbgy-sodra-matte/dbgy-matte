worked_example: |
  ## Tre nya ekvationstyper

  ### 1. Andragradsekvation via faktorisering

  Tre steg:
  1. Allt på en sida (= 0)
  2. Faktorisera
  3. Nollproduktregeln

  ```
  x² + 3x = 10
  x² + 3x − 10 = 0          ← allt på en sida
  (x + 5)(x − 2) = 0        ← faktorisera (sök tal med produkt −10, summa 3)
  x = −5 eller x = 2
  ```

  ### 2. Variabel i nämnaren — multiplicera bort

  ```
  6/(x − 1) = 3
  6 = 3(x − 1)              ← multiplicera båda leden med (x − 1)
  6 = 3x − 3
  3x = 9
  x = 3
  ```

  **Kontroll:** 6/(3 − 1) = 6/2 = 3 ✓
  Och x = 3 gör inte nämnaren 0 — lösningen är giltig.

  ### 3. Ekvation med kvadratrot — kvadrera båda leden

  ```
  √x + 2 = 5
  √x = 3                    ← isolera roten först
  x = 9                     ← kvadrera båda leden
  ```

  **Kontroll:** √9 + 2 = 3 + 2 = 5 ✓

  > **Varning:** kvadrering kan introducera "falska" lösningar. Kontrollera alltid svaret i originalekvationen.

  ## Strategi

  1. Identifiera typ — finns nämnare, rot eller andragrad?
  2. Eliminera nämnare/rot FÖRST (multiplicera/kvadrera)
  3. Allt på en sida vid andragrad
  4. Faktorisera och nollproduktregel
  5. Kontrollera

## Fler ekvationstyper

I 1.2a och 1.3 lärde vi oss linjära ekvationer och ekvationer med nämnare. I 4.2 utvidgade vi. Här fördjupar vi ytterligare med:

- **Andragradsekvationer via faktorisering** (knyter ihop 4.1 och 4.5)
- **Ekvationer med variabel i nämnaren**
- **Ekvationer med kvadratrot**

## Andragradsekvation — full process

Tre steg:

1. **Allt på en sida**, så ekvationen blir = 0.
2. **Faktorisera** (bryt ut, konjugatregeln, eller hitta två tal).
3. **Nollproduktregeln** — sätt varje faktor = 0 och lös.

> *Lös: x² + 3x = 10*

```
x² + 3x − 10 = 0          (allt på en sida)
(x + 5)(x − 2) = 0        (faktorisera)
x = −5 eller x = 2        (nollproduktregeln)
```

## Variabel i nämnaren

**Trick:** multiplicera båda leden med nämnaren.

> *Lös: 6/(x − 1) = 3*

```
6 = 3(x − 1)
6 = 3x − 3
9 = 3x
x = 3
```

**Kontroll:** sätt in i originalekvationen. 6/(3 − 1) = 3 ✓

**Varning:** kontrollera alltid att lösningen inte gör nämnaren = 0 (då är ekvationen odefinierad).

## Ekvationer med kvadratrot

**Trick:** isolera kvadratroten, sedan kvadrera båda leden.

> *Lös: √x + 2 = 5*

```
√x = 3            (isolera roten)
(√x)² = 3²        (kvadrera)
x = 9
```

**Kontroll:** √9 + 2 = 3 + 2 = 5 ✓

**Viktig fälla:** kvadrering kan introducera "falska" lösningar — kontrollera alltid!

## Strategi

1. **Identifiera ekvationstypen.**
2. **Eliminera nämnare/rötter** först om de finns.
3. **Allt på en sida** vid andragrad.
4. **Faktorisera och nollproduktregel** vid andragrad.
5. **Kontrollera** i originalekvationen.
