worked_example: |
  ## Kvadreringsregeln — utantill-formler

  ```
  (a + b)² = a² + 2ab + b²
  (a − b)² = a² − 2ab + b²
  ```

  Skillnaden är bara tecknet på mellantermen (`2ab`).

  ### Exempeltabell

  | Uttryck | Utvecklat |
  |---|---|
  | (x + 3)² | x² + 6x + 9 |
  | (x − 2)² | x² − 4x + 4 |
  | (x + 5)² | x² + 10x + 25 |
  | (2x + 1)² | 4x² + 4x + 1 |

  > **Vanlig fälla:** (x + 3)² är **inte** x² + 9. Mellantermen 2·x·3 = 6x får inte glömmas.

  ## Konjugatregeln (samma som faktorisering, fast omvänt)

  ```
  (a + b)(a − b) = a² − b²
  ```

  ### Exempel

  - (x + 4)(x − 4) = x² − 16

  ## Förenkling med flera parenteser

  ```
  3(x + 2) − (x − 5)
  = 3x + 6 − x + 5      ← multiplicera in och vänd tecken
  = 2x + 11
  ```

  > **Var noga:** `−(x − 5) = −x + 5` (vänder båda tecknen).

  ## Skillnad av två kvadrater

  ```
  (x + 3)² − (x − 2)²
  = (x² + 6x + 9) − (x² − 4x + 4)
  = x² + 6x + 9 − x² + 4x − 4
  = 10x + 5
  ```

  De kvadratiska termerna (x²) tog ut varandra. Det händer ofta — utnyttja det.

  ## Förenkla bråk med variabler

  Faktorisera täljare/nämnare. Stryk gemensamma faktorer.

  ```
  (x² − 4) / (x − 2)
  = (x − 2)(x + 2) / (x − 2)       ← konjugatregeln på täljaren
  = x + 2          (för x ≠ 2)
  ```

## Kvadreringsregeln — ett kraftfullt verktyg

Två formler du bör kunna utantill:

```
(a + b)² = a² + 2ab + b²
(a − b)² = a² − 2ab + b²
```

Skillnaden: tecknet på mellantermen (`2ab`).

### Härledning

(a + b)² betyder (a + b)(a + b). Om vi multiplicerar in:

```
(a + b)(a + b) = a·a + a·b + b·a + b·b
              = a² + ab + ab + b²
              = a² + 2ab + b²
```

På samma sätt med minus.

### Exempel

| Uttryck | Utvecklat |
|---|---|
| (x + 3)² | x² + 6x + 9 |
| (x − 2)² | x² − 4x + 4 |
| (x + 5)² | x² + 10x + 25 |
| (2x + 1)² | 4x² + 4x + 1 |
| (x − 7)² | x² − 14x + 49 |

**Tips:** mellantermen är alltid 2 · (talet före x) · (talet i parentesen).

## Vanlig fälla — (a + b)² ≠ a² + b²!

> *(x + 3)² är **inte** x² + 9.*

Mellantermen (2·x·3 = 6x) är **viktig** och får inte glömmas.

## Konjugatregeln — som vi mötte i förra lektionen

```
(a + b)(a − b) = a² − b²
```

Den används för att **multiplicera in** men är också nyckeln till **faktorisering**.

> *(x + 4)(x − 4) = x² − 16*

## Förenkling steg-för-steg

> *Förenkla: 3(x + 2) − (x − 5)*

```
3(x + 2) − (x − 5)
= 3x + 6 − x + 5    ← multiplicera in (var noga med tecknen!)
= 2x + 11
```

> *Förenkla: (x + 3)² − (x − 2)²*

```
= (x² + 6x + 9) − (x² − 4x + 4)
= x² + 6x + 9 − x² + 4x − 4
= 10x + 5
```

(De kvadratiska termerna tog ut varandra. Det händer ofta — utnyttja det!)

## Förenkla bråk med variabler

När både täljare och nämnare har gemensamma faktorer kan de strykas.

> *Förenkla: (x² − 9) / (x + 3)*

Faktorisera täljaren med konjugatregeln:
```
x² − 9 = (x − 3)(x + 3)
```

Då:
```
(x − 3)(x + 3) / (x + 3) = x − 3
```

(för x ≠ −3 — annars division med noll)

## Strategi-checklista

1. **Multiplicera in i alla parenteser** — var noga med tecknen
2. **Använd kvadreringsregeln** vid (a ± b)²
3. **Använd konjugatregeln** vid (a + b)(a − b)
4. **Samla samma sort** — x-termer, x²-termer, konstanter
5. **Vid bråk**: faktorisera, stryk gemensamma

## Tips

- **Kvadrering har TRE termer** (a², 2ab, b²). Glöm aldrig mellantermen.
- **Bromsen är tecknen.** Ett misstag och hela förenklingen blir fel.
- **Kontrollera** genom att sätta in ett enkelt x-värde i original OCH i ditt förenklade svar — de bör ge samma värde.
