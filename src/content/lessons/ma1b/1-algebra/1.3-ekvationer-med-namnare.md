  | Steg | Vad vi gör | Varför |

  Hela parentesen är "över strecket". Behandla den som EN sak. Multiplicera båda leden med 2:

  ```
  (x + 3)/2 = 8
  x + 3 = 16        ← multiplicera med 2

  ```

  ```
  x/2 + x/4 = 9         | · 4
  2x + x = 36
  3x = 36
  x = 12
  ```

## Vad är en nämnare?

I bråket `x/3` har vi två delar:

- **Täljare** = x (det som står *över* strecket)
- **Nämnare** = 3 (det som står *under* strecket)

`x/3` betyder **"x delat på 3"**. Om x är 12 så är x/3 = 12/3 = 4.

## Ekvationer med nämnare — hela tricket

Hela idén: om en nämnare står i vägen — **multiplicera bort den**.

Det funkar för att multiplikation är **motsatsen** till division. Om vi har x delat på 3 och multiplicerar med 3, så tar de ut varandra:

```
3 · x/3 = x
```

Och vi får tillbaka x ensamt. Som vanligt: det vi gör på ena sidan måste vi göra på den andra.

## Worked example — lös x/4 = 5

| Steg | Vad vi gör | Varför |
|---|---|---|
| `x/4 = 5` | Starta | — |
| `x = 5 · 4` | Multiplicera båda leden med 4 | Bli av med nämnaren |
| **`x = 20`** | Klart | — |

**Kontroll:** 20/4 = 5 ✓

## När det finns en parentes i täljaren

`(x + 3)/2 = 8` — vad gör vi?

Hela parentesen är i täljaren. Behandla den som EN sak. Multiplicera båda leden med 2:

```
(x + 3)/2 = 8
x + 3 = 16        ← multiplicera med 2
x = 13
```

## När det finns flera bråk med olika nämnare

`x/2 + x/4 = 9` — knepigare.

Trick: hitta en **gemensam nämnare**. Det är ett tal som båda nämnarna går jämnt upp i.

- 2 går i: 2, 4, 6, 8, …
- 4 går i: 4, 8, 12, …

Det första som finns i båda listorna är **4**. Det är gemensam nämnare.

Skriv om alla bråk så de har samma nämnare:

```
x/2 = 2x/4      (förlängde med 2 i både täljare och nämnare)
x/4 = x/4       (oförändrat)
```

Nu kan vi addera täljarna:

```
2x/4 + x/4 = 3x/4
```

Och lösa: `3x/4 = 9 → 3x = 36 → x = 12`.

**Alternativ metod (ofta enklare):** multiplicera HELA ekvationen med gemensam nämnare från början. Då försvinner alla nämnare på en gång.

```
x/2 + x/4 = 9         | · 4
4 · x/2 + 4 · x/4 = 4 · 9
2x + x = 36
3x = 36
x = 12
```

Det är ofta den snabbaste vägen. **Multiplicera alla termer med gemensam nämnare — då blir det inga bråk kvar.**

## När x finns i nämnaren

`6/x = 2` — ovanligare men viktigt.

Här står x i nämnaren. Vi vill bli av med "delat på x". Multiplicera båda leden med x:

```
6/x = 2
6 = 2x        ← multiplicera med x
x = 3         ← dela med 2
```

**Viktigt att veta:** x får aldrig vara 0 i en sådan ekvation, för då skulle vi dela med noll (vilket är odefinierat). Om du löser och får x = 0 är något fel.

## Strategin — sammanfattad

1. **Identifiera nämnarna** i ekvationen
2. **Hitta gemensam nämnare** om det finns flera olika
3. **Multiplicera alla termer** med gemensam nämnare — då försvinner bråken
4. **Lös som en vanlig ekvation**
