worked_example: |
  Potensekvationer är ekvationer där x är upphöjt till något. Det viktiga är att hålla koll på **antalet lösningar**.

  ## 1. `x² = a` — TVÅ lösningar

  ```
  x² = 9
  ```

  Vilket tal upphöjt till 2 blir 9?
  - 3² = 9 ✓
  - (−3)² = 9 ✓ (minus · minus = plus)

  **Två svar:** x = 3 eller x = −3. Skrivs ofta som **x = ±3**.

  > Ekvationen x² = a (med a > 0) har ALLTID två lösningar.

  ## 2. `x³ = a` — EN lösning

  ```
  x³ = 8
  ```

  - 2 · 2 · 2 = 8 ✓
  - (−2)³ = −8 ✗ (minus · minus · minus = minus)

  **En lösning:** x = 2.

  ## 3. När det inte går jämnt — förenkla med kvadratrot

  ```
  x² = 18
  x = ±√18 = ±√(9 · 2) = ±3√2
  ```

  Trick: hitta största kvadrattalet som är faktor i talet. Plocka ut det ur roten.

  ## 4. När problemet kommer från verkligheten

  | Situation | Ekvation | Vad gäller |
  |---|---|---|
  | Kvadrats area = 64 | s² = 64 | s = ±8, **men sida ≥ 0** → s = 8 |
  | Kubs volym = 125 | s³ = 125 | s = 5 |

  > **Tolka tillbaka.** Matematiskt finns två lösningar — men i verkligheten gäller bara den som är fysiskt möjlig.

## Vad är en potensekvation?

En **potensekvation** är en ekvation där det okända (vanligtvis x) är upphöjt till en potens.

| Exempel | Vad det betyder |
|---|---|
| `x² = 36` | x² = 36 |
| `x³ = 8` | x · x · x = 8 |
| `2x² = 50` | 2 · x² = 50 |

Vårt jobb: hitta vilket värde på x som gör ekvationen sann.

## x² = a — två lösningar!

Det här är annorlunda än linjära ekvationer:

```
x² = 9
```

Vilket tal upphöjt till 2 blir 9?
- `3² = 9` ✓
- `(−3)² = 9` ✓ (eftersom minus · minus = plus)

**Två svar:** x = 3 *eller* x = −3. Skrivs ofta `x = ±3` (plus eller minus 3).

> **Regel:** ekvationen `x² = a` (där a > 0) har alltid TVÅ lösningar — den positiva och den negativa.

Vad om `x² = 0`? Då finns bara EN lösning: x = 0.

Vad om `x² = −9`? Då finns **ingen** lösning — inget reellt tal blir negativt när vi upphöjer det till 2. (Det finns "imaginära tal" som löser det, men dem lär du dig senare.)

## Kvadratroten √

Tecknet `√` betyder **kvadratrot** — det positiva talet som upphöjs till 2 ger talet under roten.

| Exempel | Svar |
|---|---|
| √9 | 3 |
| √36 | 6 |
| √100 | 10 |
| √2 | ≈ 1,41 (oändligt decimalbråk) |

**Viktigt:** `√9 = 3`, *inte* `±3`. Kvadratrots-symbolen ger bara det positiva svaret. Men när vi *löser ekvationen* `x² = 9` får vi båda lösningarna.

## När det inte går jämnt ut

Vad är x i `x² = 18`?

Det finns inget heltal som upphöjt till 2 ger 18. Då skriver vi svaret **exakt** med kvadratrot:

```
x = ±√18
```

Det kan ofta förenklas:

```
√18 = √(9 · 2) = √9 · √2 = 3√2
```

(Den enklare faktorn — 9 — har en exakt kvadratrot, 3, så vi "lyfter ut" den.)

Som decimaltal: x ≈ ±4,24.

## x³ = a — bara EN lösning

För kubikekvationer är det annorlunda:

```
x³ = 8
```

`2³ = 2 · 2 · 2 = 8` ✓
`(−2)³ = −8` ❌ (minus gånger minus gånger minus = minus)

Bara EN lösning: x = 2.

**Regel:** ekvationer på formen `x^n = a` har:
- **Två lösningar** om n är jämn och a är positiv (t.ex. x² = 4, x⁴ = 16)
- **En lösning** om n är udda (t.ex. x³ = 27, x⁵ = 32)
- **Ingen lösning** om n är jämn och a är negativ (t.ex. x² = −9)

## När x² inte är ensam

`3x² = 75` — vad gör vi?

Få x² ensam först:

```
3x² = 75
x² = 25         | dela med 3
x = ±5
```

Strategin är samma som för linjära ekvationer: **få x²-termen ensam, sedan ta roten ur.**

## Tolka svaret i verkligheten

Många potensekvationer kommer från **areor och volymer**.

> *En kvadrat har arean 49 m². Hur lång är sidan?*

`s² = 49 → s = ±7`. Men en sida kan inte vara negativ! **Svar: s = 7 m.**

**Regel:** matematiskt finns två lösningar, men i en verklig situation kan bara den ena vara meningsfull. Kolla alltid tillbaka i ursprungsfrågan.
