worked_example: |
  > **Två kompisar, Maja och Noa, ska köpa pizza tillsammans. Maja betalar dubbelt så mycket som Noa. Tillsammans betalar de 240 kr. Hur mycket betalar Noa?**

  ## Steg 1 — välj variabel

  Vad är okänt? Båda beloppen. Men det räcker med EN variabel — välj den enklaste:

  > Låt **x = Noas belopp** i kr.

  ## Steg 2 — översätt mening för mening

  - "Maja betalar dubbelt så mycket som Noa" → Maja betalar **2x**
  - "Tillsammans 240" → **x + 2x = 240**

  ## Steg 3 — lös ekvationen

  | Rad | |
  |---|---|
  | `x + 2x = 240` | Ekvationen |
  | `3x = 240` | Samla x-termerna |
  | **`x = 80`** | Dela båda leden med 3 |

  ## Steg 4 — tolka tillbaka

  Noa betalar **80 kr**. Maja betalar 2 · 80 = **160 kr**.

  **Kontroll:** 80 + 160 = 240 ✓

  > **Hela tricket:** uttryck ALLA okända med samma x. Välj den ena som "grunden" — de andra skrivs som uttryck i x.

        Samma två internetabonnemang som i C-uppgiften ovan (A: 50 kr fast + 4 kr/h TV + 10 kr/h spel. B: 200 kr fast + 4 kr/h TV + 4 kr/h spel).

        a) Skriv en formel för totalkostnaden T för varje abonnemang, där t = antal timmar TV och s = antal timmar spel.

        b) Vid hur många timmars speltid (s) kostar abonnemangen lika mycket — och påverkar TV-tiden (t) svaret?

        (Öppet svar — skriv båda formlerna OCH räkningen som ger när de blir lika.)
      hint1: "a) Mönstret är T = fast del + (per-enhet · antal enheter) för varje typ av tid. b) Sätt formlerna lika och se vad som händer med t-termerna."
      hint2: "a) A: T = 50 + 4t + 10s. B: T = 200 + 4t + 4s. b) Sätt T_A = T_B och förenkla — vad händer med 4t på båda sidor?"
      hint3: "När du sätter T_A = T_B blir det 50 + 4t + 10s = 200 + 4t + 4s. 4t finns på BÅDA sidor — det kan subtraheras bort. Då har du en ekvation som bara beror på s. Lös ut s."
      solution: |
        ## a) Formler

        **A:** T = 50 + 4t + 10s
        **B:** T = 200 + 4t + 4s

        ## b) När lika kostnad?

        Sätt T_A = T_B:

        ```
        50 + 4t + 10s = 200 + 4t + 4s
        50 + 10s = 200 + 4s        ← 4t tar ut sig själv på båda sidor
        10s − 4s = 200 − 50
        6s = 150
        s = 25 timmar
        ```

        **Vid 25 timmars speltid kostar abonnemangen lika.** TV-tiden påverkar inte svaret eftersom kostnaden per TV-timme (4 kr) är densamma i båda abonnemangen — den faller bort när vi sätter formlerna lika.

        > **Pedagogisk insikt:** när en variabel har samma koefficient i båda jämförda formler kan vi alltid räkna utan den.

        Låt x = Annas sträcka (i meter).
        Bo: x/2 (hälften av Anna)
        Carl: 3(x + x/2) = 3 · 3x/2 = 9x/2

        ## Ekvation

        x + x/2 + 9x/2 = 900

        Lägg ihop: x = 2x/2, så vänster sida blir 2x/2 + x/2 + 9x/2 = 12x/2 = 6x.

        ```
        6x = 900
        x = 150
        ```

        - **Anna:** 150 m
        - **Bo:** 150/2 = 75 m
        - **Carl:** 9 · 150 / 2 = 675 m

        Förklara matematiskt varför följande ekvation saknar lösning:

        4(x − 2) + 3 = 4x + 1

        (Skriv din förklaring i textrutan och jämför sedan med lösningen.)
      hint1: "Konceptet: ibland leder en ekvation till ett påstående som är omöjligt. Det betyder att inget värde på x kan lösa den. Börja med att förenkla båda leden."
      hint2: "Strategi: vänster sida blir 4(x − 2) + 3 = 4x − 8 + 3 = 4x − 5. Höger sida är 4x + 1. Ekvationen blir 4x − 5 = 4x + 1. Vad händer om du försöker lösa den?"
      hint3: "Subtrahera 4x från båda leden. Vad blir kvar? Vad innebär det att leden inte är lika — kan ekvationen lösas då?"
      solution: |
        Förenkla båda leden:

        Vänster: 4(x − 2) + 3 = 4x − 8 + 3 = 4x − 5
        Höger: 4x + 1

        Ekvationen blir alltså: 4x − 5 = 4x + 1

        Subtrahera 4x från båda leden:
        −5 = 1

        Detta är falskt. Eftersom vi gjort korrekta omformningar och ändå landat i ett falskt påstående, finns det **inget värde på x** som kan lösa ekvationen.

## Från text till ekvation

Många riktiga matematik-problem kommer som **berättelser** — inte som färdiga uttryck. Den verkliga konsten är att översätta texten till en ekvation. Sedan är resten bara teknik.

**Tre steg som alltid fungerar:**

1. **Välj variabeln.** Vad är okänt? Bestäm: "låt x = …" och skriv ner vad x står för.
2. **Översätt meningen** del för del. Skriv om vart och ett av textens nyckelorden till matematiska symboler.
3. **Lös ekvationen och tolka tillbaka.** När du hittat x — vad betyder det i originalsituationen?

## Översättningstabell

| Vardagsord | Matematik |
|---|---|
| "är", "blir" | = |
| "summan av" | + |
| "skillnaden mellan" | − |
| "det dubbla av", "två gånger" | 2x |
| "hälften av" | x/2 |
| "5 mer än" | x + 5 |
| "3 mindre än" | x − 3 |
| "tre på varandra följande" | x, x+1, x+2 |
| "tillsammans" | summan |

## Worked example — översättning steg för steg

**Problem:**
> Två kompisar, Maja och Noa, ska köpa pizza tillsammans. Maja betalar dubbelt så mycket som Noa. Tillsammans betalar de 240 kr. Hur mycket betalar Noa?

**Steg 1 — välj variabel:**
Vad är okänt? Båda beloppen. Men det räcker med en variabel — välj den enklaste:
> Låt x = Noas belopp i kr.

**Steg 2 — översätt:**
- Maja betalar dubbelt så mycket som Noa → Maja betalar 2x
- Tillsammans 240 → x + 2x = 240

**Steg 3 — lös:**

| Steg | |
|---|---|
| `x + 2x = 240` | Ekvationen |
| `3x = 240` | Samla |
| `x = 80` | Dela med 3 |

**Tolka tillbaka:** Noa betalar 80 kr. Maja betalar 2 · 80 = 160 kr. (Kontroll: 80 + 160 = 240 ✓)

## När det finns flera okända

Trick: **uttryck alla okända med samma variabel**. Välj den ena som x, sedan skriv ner de andra som uttryck i x.

> *Sara har dubbelt så mycket som Adam. Mira har 1 500 kr mindre än Sara.*

Bästa val: låt **Adams** belopp vara x (det är "grunden" som de andra mäts mot).

- Adam = x
- Sara = 2x
- Mira = 2x − 1500

Då har vi tre okända med EN variabel. När x är löst får vi alla tre.

## Jämföra två alternativ

Många praktiska problem handlar om att jämföra: vilket är billigast? Vid vilken punkt är de lika?

**Tekniken:**
1. Skriv en formel för varje alternativ
2. Sätt formlerna lika med varandra (`K_A = K_B`)
3. Lös ekvationen — det säger vid vilket värde alternativen är lika
4. Undersök vad som händer "före" och "efter" det värdet

## Resonemang — när en ekvation saknar lösning

Ibland ger en korrekt utförd ekvationslösning ett **omöjligt påstående**, som `5 = 7`. Det betyder att ekvationen **saknar lösning** — inget värde på x kan göra båda leden lika.

Geometriskt: ekvationen `4x − 5 = 4x + 1` är två räta linjer med **samma lutning** men olika nivåer. Parallella linjer möts aldrig — därför finns ingen lösning.

(Det finns också ekvationer som är *sanna för alla x*, t.ex. `2(x + 1) = 2x + 2`. De har oändligt många lösningar.)
