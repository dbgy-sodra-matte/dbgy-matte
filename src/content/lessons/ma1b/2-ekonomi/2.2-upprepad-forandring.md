worked_example: |
  Den **utvidgade formeln** för förändring över tid:

  ```
  slutvärde = startvärde · (förändringsfaktor)^n
  ```

  där `n` är antalet förändringar (oftast år).

  ## 1. Sparkonto med ränta — `5 000 kr i 3 år med 2 %`

  ```
  slutvärde = 5 000 · 1,02³
            = 5 000 · 1,0612...
            ≈ 5 306 kr
  ```

  ## 2. Bil tappar värde — `80 000 kr, −20 % per år i 2 år`

  ```
  slutvärde = 80 000 · 0,80²
            = 80 000 · 0,64
            = 51 200 kr
  ```

  ## 3. Olika förändringsfaktorer per år

  Om förändringsfaktorn varierar — multiplicera bara dem i ordning:

  ```
  Mobil 12 000 kr, −30 % år 1, −20 % år 2 och 3:
  värde efter 3 år = 12 000 · 0,70 · 0,80 · 0,80
                   = 12 000 · 0,448
                   = 5 376 kr
  ```

  ## 4. 72-regeln — snabb-uppskattning för fördubbling

  > **Vid en ränta r % tar det ungefär `72/r` år innan pengarna fördubblas.**

  | Ränta | Fördubblingstid |
  |---|---|
  | 2 % | ~36 år |
  | 4 % | ~18 år |
  | 6 % | ~12 år |
  | 8 % | ~9 år |

## När något ändras flera gånger

I förra lektionen lärde vi oss att räkna ut en enskild förändring:

```
nytt värde = gammalt värde · förändringsfaktor
```

Men i verkligheten ändras saker ofta **flera gånger i rad**.

- Ett sparkonto med ränta varje år
- En bil som tappar värde varje år
- En befolkning som växer årligen

Här kommer den **utvidgade formeln**:

```
slutvärde = startvärde · (förändringsfaktor)^n
```

där `n` är antalet förändringar (oftast antalet år).

## Worked example — sparande med ränta

Du sätter in **3 000 kr** på ett konto med **4 % årlig ränta**. Hur mycket har du efter 5 år?

```
Förändringsfaktor: +4 % → 1,04
Antal år: 5
Slutvärde = 3 000 · 1,04⁵
```

Räkna ut `1,04⁵`:

```
1,04 · 1,04 = 1,0816
1,0816 · 1,04 = 1,1249...
1,1249 · 1,04 = 1,1699...
1,1699 · 1,04 = 1,2167
```

(eller använd räknarens potensknapp `^`)

```
Slutvärde = 3 000 · 1,2167 ≈ 3 650 kr
```

Du har tjänat ca **650 kr** på 5 år bara genom att låta pengarna stå.

## Worked example — värdeminskning

En begagnad cykel köps för **8 000 kr**. Den tappar **10 % av värdet per år**. Vad är värdet efter 4 år?

```
Förändringsfaktor: −10 % → 0,90
Slutvärde = 8 000 · 0,90⁴
         = 8 000 · 0,6561
         = 5 249 kr
```

Cykeln är värd ca **5 250 kr** efter 4 år.

## Vad om förändringsfaktorn ändras?

Ibland är minskningen olika år för år. Då bara multiplicerar du med rätt faktor varje år.

> *En mobiltelefon tappar 30 % första året och 20 % varje år efter det. Värdet efter 3 år (startpris 12 000 kr)?*

```
12 000 · 0,70 · 0,80 · 0,80 = 12 000 · 0,448 = 5 376 kr
```

Ingen magi — bara en multiplikation per förändring.

## Ränta-på-ränta — den "magiska" tillväxten

Albert Einstein lär ha sagt: "Ränta-på-ränta är världens åttonde underverk."

Vad menade han? Att pengar som lämnas att växa på sig själv ökar **exponentiellt** — inte linjärt.

| År | Pengar (med 5 % ränta från 10 000 kr) |
|---|---|
| 0 | 10 000 |
| 5 | 12 763 |
| 10 | 16 289 |
| 20 | 26 533 |
| 30 | 43 219 |
| 40 | 70 400 |

Notera: på 40 år har pengarna ökat med **60 000 kr** — utan att du gjort något.

## 72-regeln — en användbar genväg

Hur lång tid tar det för pengar att **fördubblas**?

```
Tid (år) ≈ 72 / ränta
```

| Ränta | Tid till fördubbling |
|---|---|
| 2 % | ca 36 år |
| 4 % | ca 18 år |
| 6 % | ca 12 år |
| 8 % | ca 9 år |
| 10 % | ca 7 år |

Bra för snabba mentala beräkningar. (Den exakta formeln kräver logaritmer som ni lär er i Ma2.)

## Strategi vid problem

1. **Identifiera startvärde och förändringsfaktor.**
2. **Antal år? Eller antal förändringar?** Det är ofta samma sak.
3. **Använd formeln:** slutvärde = startvärde · (förändringsfaktor)^n
4. **Pröva med olika n** om problemet handlar om hur lång tid något tar.
5. **Tolka tillbaka i situationen.** Är svaret rimligt?
