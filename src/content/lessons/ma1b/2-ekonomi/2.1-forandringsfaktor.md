worked_example: |
  Hela tricket: **nytt värde = gammalt värde · förändringsfaktor**.

  ## 1. Ökning — `+15 % på 200 kr`

  | Steg | |
  |---|---|
  | Räkna ut faktorn | +15 % → **1,15** |
  | Multiplicera | 200 · 1,15 = **230 kr** |

  **Kontroll:** 15 % av 200 är 30 kr. 200 + 30 = 230 ✓

  > **Tumregel ökning:** 1 + procenttalet som decimal.

  ## 2. Minskning — `−40 % på 600 kr`

  | Steg | |
  |---|---|
  | Räkna ut faktorn | −40 % → **0,60** (du har KVAR 60 %) |
  | Multiplicera | 600 · 0,60 = **360 kr** |

  > **Tumregel minskning:** 1 − procenttalet som decimal.

  ## 3. Baklänges — `4 500 kr efter −30 %, vad var ursprunget?`

  Vi vet nya priset. Vi vet faktorn. Vi söker det gamla. **Dela** istället för att multiplicera.

  ```
  gammalt · 0,70 = 4 500
  gammalt = 4 500 / 0,70 ≈ 6 429 kr
  ```

  ## 4. Hur många procent? — `från 250 kr till 290 kr`

  | Steg | |
  |---|---|
  | Räkna faktorn | 290 / 250 = **1,16** |
  | Subtrahera 1 | 1,16 − 1 = 0,16 = **16 %** ökning |

  > **Sammanfattning:** ökning → 1+x. Minskning → 1−x. Baklänges → dela. Procent från två värden → nytt/gammalt sen minus 1.

## Vad är en förändringsfaktor?

Tänk dig att du vet vad ett pris kostar idag, och du vill räkna ut det nya priset efter en procentuell ändring. Förändringsfaktorn är **det tal du multiplicerar med**.

**Formeln:**
```
nytt värde = gammalt värde · förändringsfaktor
```

Det är hela hemligheten.

## Hur räknar man ut förändringsfaktorn?

Två fall:

### Ökning

| Ökning | Förändringsfaktor |
|---|---|
| +5 % | 1,05 |
| +10 % | 1,10 |
| +25 % | 1,25 |
| +100 % | 2,00 (priset dubblas) |

**Tumregel:** lägg 1 framför decimalformen av procenttalet.
+12 % → 1,12. +37 % → 1,37.

Varför? För att du behåller hela det gamla värdet (100 % = 1) och lägger till ökningen.

### Minskning

| Minskning | Förändringsfaktor |
|---|---|
| −5 % | 0,95 |
| −10 % | 0,90 |
| −25 % | 0,75 |
| −50 % | 0,50 (priset halveras) |

**Tumregel:** ta 1 minus decimalformen.
−12 % → 1 − 0,12 = 0,88. −30 % → 0,70.

Varför? Du behåller en del — det som blir KVAR efter minskningen. Tar du bort 25 %, behåller du 75 %.

## Worked example — ökning

En t-shirt kostar 200 kr. Priset höjs med 15 %. Vad är det nya priset?

```
Förändringsfaktor: +15 % → 1,15
Nytt pris = 200 · 1,15 = 230 kr
```

**Kontroll:** 15 % av 200 är 30 kr. 200 + 30 = 230 ✓

## Worked example — minskning

En kapuschong kostar 600 kr. Den säljs på rea med 40 % rabatt. Vad är reapriset?

```
Förändringsfaktor: −40 % → 0,60
Reapris = 600 · 0,60 = 360 kr
```

**Kontroll:** 40 % av 600 är 240 kr. 600 − 240 = 360 ✓

## "Baklänges" — hitta det gamla värdet

Ibland vet vi det NYA värdet och vill räkna ut det GAMLA.

> *En cykel kostar 4 500 kr efter en rea på 30 %. Vad kostade den innan?*

Sätt upp en ekvation:

```
gammalt · 0,70 = 4 500
gammalt = 4 500 / 0,70 = 6 428,57 kr
```

Det gamla priset var ca **6 429 kr**.

**Tumregel:** för att räkna baklänges — **dela** med förändringsfaktorn istället för att multiplicera.

## "Hur mycket har det ändrats?" — räkna procent från två värden

Givet två tal — gammalt och nytt — kan du räkna ut den procentuella förändringen.

> *En aktie ökar från 250 kr till 290 kr. Med hur många procent har den ökat?*

Steg:
1. Räkna ut förändringsfaktorn: nytt / gammalt = 290 / 250 = **1,16**
2. Subtrahera 1: 1,16 − 1 = **0,16 = 16 %**

Aktien har ökat med **16 %**.

## Vad det här kommer användas till

Förändringsfaktorn är hjärtat i ekonomi-momentet. Den används i:
- Reapris och prishöjningar (denna lektion)
- Upprepade förändringar — när något ändras flera gånger (nästa lektion)
- Ränta och lån (om en vecka)
