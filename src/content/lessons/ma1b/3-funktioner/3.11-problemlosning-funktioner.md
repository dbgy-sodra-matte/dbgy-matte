worked_example: |
  Strategi i tre steg — fungerar för alla problem:

  ## Steg 1 — vilken funktionstyp?

  | Ord i texten | Funktionstyp |
  |---|---|
  | "per styck", "+ X kr/timme" + fast del | **Linjär** |
  | "X % per år", "fördubblas", "halveras" | **Exponential** |
  | "area", "volym", "kvadrat", "kub" | **Potens** |
  | "tappar X % per år" | **Exponential (avtagande)** |

  ## Steg 2 — hitta parametrarna

  - **Linjär (y = kx + m):** m = fast del, k = per-enhet
  - **Exponential (y = C · a^x):** C = startvärde, a = förändringsfaktor
  - **Potens (y = C · x^a):** ofta given av en geometrisk/fysikalisk formel

  ## Steg 3 — använd funktionen

  | Fråga | Vad du gör |
  |---|---|
  | "Värdet efter X tid?" | Sätt in x i funktionen |
  | "När är värdet exakt Y?" | Lös ekvationen f(x) = Y |
  | "När är värdet över/under Y?" | Pröva olika x (för exponential) |
  | "När är två alternativ lika?" | Sätt funktionerna lika, lös för x |

  ## Worked example — taxi

  > Taxi: 60 kr startavgift + 15 kr/km. Hur långt har du åkt om kostnaden är 180 kr?

  Funktion: K(s) = 60 + 15s (linjär).

  Sätt K = 180:
  ```
  60 + 15s = 180
  15s = 120
  s = 8 km
  ```

  ## Worked example — pröva exponential

  > Bakterier dubblas varje timme. Start: 200. Vid vilket t är antalet över 10 000?

  Funktion: B(t) = 200 · 2^t.

  Söker minsta t där 2^t > 50:
  - 2⁵ = 32 (under)
  - 2⁶ = 64 (över!)

  Efter **6 timmar**.

  > **Pröva-strategin:** för exponentialfunktioner saknar vi en exakt formel för att lösa baklänges. Pröva olika t i kalkylator tills du hittar gränsen.

## Tre steg till lösning

Varje funktions-problem följer samma tre steg:

1. **Identifiera funktionstypen.** Linjär, exponential eller potens?
2. **Skriv funktionen.** Med rätt parametrar (k, m, C, a, ...).
3. **Använd funktionen** för att svara på frågan.

## Steg 1 — Vilken funktionstyp?

| Ord/begrepp i texten | Funktionstyp |
|---|---|
| "per styck", "per timme", "per km" + fast del | **Linjär** |
| "ränta", "procentuell ökning per år" | **Exponential** |
| "fördubblas", "halveras" per period | **Exponential** |
| "area", "volym", "kvadrat", "kub" | **Potens** |
| "tappar X % per år" | **Exponential (avtagande)** |

## Steg 2 — Hitta parametrarna

**Linjär (y = kx + m):**
- m = fast/grunddel
- k = per-enhet-värde

**Exponential (y = C · a^x):**
- C = startvärde
- a = förändringsfaktor (+5 % → 1,05; −10 % → 0,90)

**Potens (y = C · x^a):**
- Beror på problemet. Ofta från fysikaliska/geometriska formler.

## Steg 3 — Använd funktionen

Vanliga frågor:

**A) "Vad är värdet efter X tid?"**
Sätt in x i funktionen.

**B) "När är värdet exakt Y?"**
Sätt funktionen lika med Y, lös för x. (För exponential — pröva olika x.)

**C) "När är värdet över/under Y?"**
Hitta brytpunkten genom att lösa f(x) = Y. Sedan kolla "före/efter".

**D) "Vid vilket värde är två alternativ lika?"**
Sätt funktionerna lika, lös för x.

## Worked example 1 — linjär

> *En löpare börjar med 5 km/vecka och ökar med 2 km varje vecka. Hur långt springer hen vecka 10?*

**Steg 1:** "+2 km varje vecka" är konstant ökning → linjär.

**Steg 2:** km(v) = 5 + 2v (m = 5 startvärde, k = 2 ökning/vecka).

**Steg 3:** km(10) = 5 + 2·10 = 25 km.

## Worked example 2 — exponential

> *Du investerar 10 000 kr med 6 % årlig avkastning. När (avrundat) har du fördubblat pengarna?*

**Steg 1:** "+6 % per år" → exponential.

**Steg 2:** K(t) = 10 000 · 1,06^t.

**Steg 3:** Vi söker t där K(t) = 20 000, alltså 1,06^t = 2.

Pröva: 1,06^10 ≈ 1,79. 1,06^12 ≈ 2,01. → ca 12 år.

(72-regeln säger: 72/6 = 12 år. ✓)

## Worked example 3 — potens (geometri)

> *En rektangulär trädgård har konstant omkrets 24 m. Sidornas längder x och y, där x + y = 12. Skriv arean A som funktion av x.*

Eftersom y = 12 − x:
```
A(x) = x · (12 − x) = 12x − x²
```

Detta är inte en ren potensfunktion utan **andragradsfunktion**.

A(2) = 2 · 10 = 20 m².
A(6) = 6 · 6 = 36 m². (Maximum — kvadrat!)
A(10) = 10 · 2 = 20 m².

## Worked example 4 — jämför två alternativ

> *Cykeluthyrning A: 100 + 15 kr/timme. Cykeluthyrning B: 60 + 20 kr/timme. När är de lika dyra?*

```
K_A(t) = 100 + 15t
K_B(t) = 60 + 20t

Sätt lika: 100 + 15t = 60 + 20t
          40 = 5t
          t = 8 timmar
```

**Tolkning:** vid 8 timmar kostar de lika (220 kr). Under 8 timmar: B är billigare. Över 8 timmar: A är billigare.

## Vanliga fällor

1. **Använda fel modell.** Procent per år är INTE linjärt.
2. **Glömma multiplicera om det är upprepad förändring.** "20 % rabatt sedan 15 % höjning" är inte −5 % netto.
3. **Tolka inte tillbaka.** Det matematiska svaret "x = 12,5" kanske ska tolkas som "13" om det handlar om besök.
4. **Bara prova ett värde.** Vid exponential måste man pröva flera för att hitta rätt.
