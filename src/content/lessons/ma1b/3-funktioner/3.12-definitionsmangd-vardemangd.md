worked_example: |
  ## Två begrepp — definitionsmängd och värdemängd

  - **Definitionsmängd** = mängden av tillåtna **x-värden** (input).
  - **Värdemängd** = mängden av möjliga **y-värden** (output).

  ## Tre regler för definitionsmängd

  1. **Division med 0 förbjudet** — nämnaren får inte vara 0.
  2. **Kvadratrot av negativt förbjudet** — uttrycket inuti roten ≥ 0.
  3. **I tillämpningar** — tid ≥ 0, antal är heltal, längd > 0, osv.

  ## Definitionsmängd — exempeltabell

  | Funktion | Begränsning | Definitionsmängd |
  |---|---|---|
  | y = x + 3 | inga | alla reella tal |
  | y = 1/x | nämnaren ≠ 0 | alla x ≠ 0 |
  | y = √x | inuti rot ≥ 0 | x ≥ 0 |
  | y = √(x − 4) | x − 4 ≥ 0 | x ≥ 4 |
  | y = 2/(x − 5) | x − 5 ≠ 0 | x ≠ 5 |

  ## Värdemängd — exempeltabell

  | Funktion | Värdemängd | Varför |
  |---|---|---|
  | y = x + 3 | alla y | linjär — alla y möjliga |
  | y = x² | y ≥ 0 | kvadrater aldrig negativa |
  | y = x² + 3 | y ≥ 3 | minst x² = 0, plus 3 |
  | y = 2^x | y > 0 | exponential alltid positiv |
  | y = √x | y ≥ 0 | rot ger icke-negativt |

  ## Tillämpningar — verklig kontra matematisk definitionsmängd

  Funktionen K(t) = 100 + 30t (cykeluthyrning) är matematiskt definierad för alla t.
  Men i verkligheten:
  - Tid är icke-negativ → t ≥ 0
  - Övre gräns beror på sammanhang (en dag = 24 h, en vecka = 168 h, osv.)

  > **Lärdom:** matematisk definitionsmängd är OFTA större än verklig. Tänk pragmatiskt.

## Definitionsmängd — vilka x-värden får vi sätta in?

**Definitionsmängden** är mängden av alla x-värden där funktionen är "definierad" (alltså ger ett meningsfullt y).

För de flesta funktioner är detta "alla reella tal". Men det finns undantag.

| Funktion | Begränsning | Definitionsmängd |
|---|---|---|
| `y = x + 3` | Inga | Alla reella tal |
| `y = x²` | Inga | Alla reella tal |
| `y = 1/x` | x ≠ 0 (annars delar vi med 0) | Alla utom 0 |
| `y = √x` | x ≥ 0 (kan inte rota negativa tal) | x ≥ 0 |
| `y = 2/(x − 5)` | x ≠ 5 (annars 0 i nämnaren) | Alla utom 5 |
| `y = √(x − 3)` | x − 3 ≥ 0 → x ≥ 3 | x ≥ 3 |

## Värdemängd — vilka y-värden kan funktionen ge?

**Värdemängden** är mängden av alla möjliga y-värden som funktionen kan producera.

| Funktion | Värdemängd | Varför |
|---|---|---|
| `y = x + 3` | Alla reella tal | Linjär funktion — alla y möjliga |
| `y = x²` | y ≥ 0 | Kvadrater är aldrig negativa |
| `y = x² + 5` | y ≥ 5 | Minst x² = 0, plus 5 |
| `y = 1/x` | y ≠ 0 | Bråk med icke-noll i täljaren är aldrig 0 |
| `y = 2^x` | y > 0 | Exponentialer är alltid positiva |
| `y = √x` | y ≥ 0 | Kvadratrot ger icke-negativt |

## I tillämpningar — den realistiska definitionsmängden

I matematik kan vi ofta ha "alla reella tal". Men i en verklig situation måste vi tänka:

- **Tid** är icke-negativ: t ≥ 0
- **Antal personer** är heltal och icke-negativ
- **Pris** är (oftast) icke-negativt
- **Längd, area, volym** är positiva
- **Procent** är ofta begränsad till 0–100 (eller 0–1 som decimal)

> *Funktion: kostnaden för en cykeluthyrning K(t) = 50 + 20t. Realistisk definitionsmängd: t ≥ 0.*

Matematiskt fungerar t = −3, men "−3 timmar" är inte meningsfullt.

## Worked example — y = x²

Funktion: `y = x²`

**Definitionsmängd:** Inga begränsningar — vi kan kvadrera vilket tal som helst. Alla reella tal.

**Värdemängd:** Kvadrater är alltid ≥ 0. Vi kan få y = 0 (vid x = 0). y kan vara hur stort som helst.

→ **Värdemängd: y ≥ 0**

## Worked example — y = √x

**Definitionsmängd:** Vi kan inte ta kvadratroten av ett negativt tal i reella matematik. → **x ≥ 0**.

**Värdemängd:** Kvadratroten ger alltid icke-negativt svar. → **y ≥ 0**.

## Worked example — y = 1/(x − 2)

**Definitionsmängd:** Nämnaren får inte vara 0. x − 2 = 0 ger x = 2. → **x ≠ 2**.

**Värdemängd:** Bråk med 1 i täljaren och icke-noll i nämnaren är aldrig 0. → **y ≠ 0**.

## Varför behöver vi tänka på det här?

Tre skäl:

**1. Undvika fel.** Om du sätter in x = 0 i y = 1/x får du fel — och om du frågar "vad är y vid x = 0" är det rätt svar att säga "det är odefinierat".

**2. Tolkningen i verkligheten.** Att svaret är "x = −3 år" kanske matematiskt rätt men praktiskt fel.

**3. Förståelse av funktioner.** Olika funktionstyper har olika "naturliga" definitionsmängder. Att se det hjälper dig se mönster.

## Notation

I gymnasiematematiken används oftast vanliga olikheter:

- `x ≥ 0` — x är större eller lika med noll
- `x > 0` — x är större än noll (men inte 0)
- `x ≠ 5` — x är inte 5
- `−2 ≤ x ≤ 10` — x ligger mellan −2 och 10 (inklusive ändpunkterna)
- `0 < x < 100` — x ligger mellan 0 och 100 (exklusive ändpunkterna)

På avancerade nivå (Ma2+) används också mängdnotation: `D = {x ∈ ℝ : x ≥ 0}`.

## Tips

- **Kolla först om något är förbjudet.** Division med 0, kvadratrot av negativt, är de vanligaste.
- **I tillämpningar — tänk realistiskt.** Tid, pris, antal måste vara icke-negativa.
- **Värdemängden ser man ofta från grafens form.** Parabel y = x² har lägsta punkt (0, 0) → värdemängd y ≥ 0.
- **Värdemängd och definitionsmängd är OLIKA saker.** Definitionsmängd handlar om x, värdemängd om y.
