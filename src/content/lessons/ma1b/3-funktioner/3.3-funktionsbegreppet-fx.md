  ## Vad betyder f(x)?

  Tre delar:
  - **f** är funktionens NAMN (kan också vara g, h, ...)
  - **x** är INPUT (det vi sätter in)
  - **f(x)** är OUTPUT (det vi får ut)

  > När du ser `f(3)` — sätt in 3 där x står i regeln.

  ## Worked example — räkna f för olika x

  Funktion: `f(x) = 2x + 1`

  | Beräkning | Resultat |
  |---|---|
  | f(3) = 2 · 3 + 1 | 7 |
  | f(0) = 2 · 0 + 1 | 1 |
  | f(−2) = 2 · (−2) + 1 | −3 |

  ## Worked example — lös f(x) = y

  Funktion: `f(x) = 4x + 7`. Vilket x ger f(x) = 23?

  ```
  4x + 7 = 23      ← sätt funktionen = 23
  4x = 16          ← subtrahera 7
  x = 4
  ```

  ## Var noga med tecknen!

  Om f(x) = x²:
  - f(−3) = (−3)² = **9**, inte −9.
  - Minus gånger minus är plus.

  ## Sätt in ett UTTRYCK (inte ett tal)

  Funktion: `f(x) = 2x + 5`. Vad är f(a + 1)?

  Sätt in `a + 1` där x står:
  ```
  f(a+1) = 2(a+1) + 5
        = 2a + 2 + 5
        = 2a + 7
  ```

  > **Trick:** allt funktionen gör med x görs nu med uttrycket du sätter in. Behandla det som en helhet.

## Vad betyder f(x)?

I förra lektionen skrev vi `y = 2x + 1`. Det är ett sätt att beskriva en funktion. Men det finns ett annat — och bättre — sätt:

```
f(x) = 2x + 1
```

Det utläses **"f av x"** eller **"funktionen f av x"**.

Vad är skillnaden mot `y = 2x + 1`?

I praktiken: ingen alls. Båda beskriver samma funktion. Men `f(x)` har två fördelar:

1. Det **markerar** vad x är — namnet "x" är synligt i (x).
2. Det **namnger** funktionen ("f") så vi kan ha **flera funktioner samtidigt**: f, g, h, …

## Att räkna ut f(något)

När du ser `f(3)` betyder det: **sätt in 3 där x står**.

Säg `f(x) = 2x + 1`:

```
f(3) = 2 · 3 + 1 = 7
```

Vi har **bytt ut x mot 3** i regeln.

### Fler exempel

| Funktion | f(3) | f(0) | f(−2) |
|---|---|---|---|
| `f(x) = x + 4` | 7 | 4 | 2 |
| `f(x) = 2x − 1` | 5 | −1 | −5 |
| `f(x) = x²` | 9 | 0 | 4 |
| `f(x) = x² + x` | 12 | 0 | 2 |

**Var noga med negativa tal.** Om f(x) = x² och du räknar f(−3):
```
f(−3) = (−3)² = 9
```
Inte −9! Minus gånger minus blir plus.

## Flera funktioner — använd olika namn

När du har flera funktioner använder du olika bokstäver:

```
f(x) = 2x + 1
g(x) = x²
h(x) = 3x − 5
```

Då kan du skriva saker som `f(4) + g(2) = 9 + 4 = 13`. Det är tydligt vilken funktion du menar.

## Att lösa ekvationer på formen f(x) = något

Ibland vill du veta: "för vilket x gäller f(x) = 11?"

Det är en vanlig ekvation. Sätt funktionen lika med 11 och lös.

> *f(x) = 3x + 2. Hitta x där f(x) = 11.*

```
3x + 2 = 11
3x = 9
x = 3
```

Kontroll: f(3) = 3 · 3 + 2 = 11 ✓

## Worked example — funktioner i text

> *En taxiresa kostar 50 kr i startavgift plus 12 kr/km. Skriv en funktion för totalkostnaden f(s) där s är antal km.*

```
f(s) = 50 + 12s
```

Funktionsnamn `f`, input `s` (antal km), regel `50 + 12s`.

Nu kan du fråga: vad kostar en 8 km resa?

```
f(8) = 50 + 12 · 8 = 50 + 96 = 146 kr
```

Eller: hur långt kan du åka för 300 kr?

```
f(s) = 300
50 + 12s = 300
12s = 250
s = 250/12 ≈ 20,8 km
```

## Notation — vad ska du inte missuppfatta?

`f(x)` är **inte** multiplikation. Det är **inte** "f gånger x". Det är **funktionens värde** för det specifika x.

| Skrivet | Betydelse |
|---|---|
| `f(x)` | Funktionen f, evaluated vid x |
| `f · x` | f gånger x (om f är ett tal) |
| `f(3)` | Funktionens värde när x = 3 |

(De flesta i matematiken skriver punktmultiplikation som `·` eller utelämnar tecknet helt: `3x` betyder `3 · x`. Och `f(x)` är funktionsnotation.)

## Tips

- **Funktionsnamnet är bara en etikett.** f, g, P, K — det är bokstaven du valt.
- **Sätt in mellan parenteserna.** Det som står i `f(...)` ersätter alla x.
- **(−3)² = 9**, inte −9.
- **Använd funktionsmaskinen** nedan för att leka — du matar in x, ser f(x), och gissar regeln.
