worked_example: |
  En **exponentialfunktion** har formen `y = C · a^x`, där:
  - **C** = startvärde (vid x = 0, eftersom a⁰ = 1)
  - **a** = förändringsfaktor (multipliceras varje steg)
  - **x** = exponenten (oftast tid)

  ## 1. Tillväxt eller avtagande?

  Avgörs av **a**:

  | a | Beteende | Exempel |
  |---|---|---|
  | a > 1 | Tillväxt | a = 1,05 → +5 % per steg |
  | a = 1 | Konstant | y ändras inte |
  | 0 < a < 1 | Avtagande | a = 0,90 → −10 % per steg |

  ## 2. Sparkonto — `5 000 kr i t år med 3 %`

  ```
  f(t) = 5 000 · 1,03^t

  Efter 10 år:
  f(10) = 5 000 · 1,03^10
        ≈ 5 000 · 1,344
        ≈ 6 720 kr
  ```

  ## 3. Värdeminskning — `bil 8 000 kr, −20 % per år`

  ```
  V(t) = 8 000 · 0,80^t

  Efter 3 år:
  V(3) = 8 000 · 0,80^3
       = 8 000 · 0,512
       = 4 096 kr
  ```

  ## 4. Halveringstid — `80 g, halveras vart 5:e år`

  Om något halveras kan vi räkna antalet halveringar som x:

  ```
  massa(t) = 80 · (1/2)^(t/5)

  På 15 år → 3 halveringar:
  80 → 40 → 20 → 10 g
  ```

  ## 5. Hitta räntan (baklänges)

  > 10 000 kr blir 14 026 kr på 5 år. Vad är räntan?

  ```
  10 000 · a^5 = 14 026
  a^5 = 1,4026
  a = 1,4026^(1/5) ≈ 1,07     ← ta femte roten
  ```

  Räntan är ca **7 %** per år.

## Vad är en exponentialfunktion?

Den har formen:

```
y = C · a^x
```

eller med funktionsnotation:

```
f(x) = C · a^x
```

Där:
- **C** är startvärdet (värdet när x = 0, eftersom a⁰ = 1)
- **a** är **förändringsfaktorn** — det vi multiplicerar med för varje steg
- **x** är exponenten (oftast tid, antal steg, antal år)

## Hur skiljer sig detta från en linjär funktion?

| | Linjär (y = kx + m) | Exponentiell (y = C · a^x) |
|---|---|---|
| Hur y ändras | Adderar samma tal per steg | Multiplicerar med samma tal per steg |
| Grafens form | Rät linje | Kurva (uppåt eller neråt) |
| Exempel | Taxiresa: +15 kr/km | Sparkonto: +5 % per år |

## Tillväxt eller avtagande?

Det avgörs av **a**:

- **a > 1** → tillväxt. y växer för varje steg.
  - Exempel: a = 1,05 (5 % ökning per steg) → tillväxt
- **a = 1** → konstant. y ändras inte.
- **0 < a < 1** → avtagande. y minskar för varje steg.
  - Exempel: a = 0,90 (10 % minskning per steg) → avtagande

Detta är samma logik som **förändringsfaktorn** vi lärde oss i Ekonomi-momentet — bara att vi här ser den som en funktion av tid.

## Worked example — sparkonto

Du sätter in **5 000 kr** på ett konto med **3 % årlig ränta**. Skriv en funktion för beloppet efter t år.

```
f(t) = 5 000 · 1,03^t
```

Vad har du efter 10 år?

```
f(10) = 5 000 · 1,03^10
      ≈ 5 000 · 1,344
      ≈ 6 720 kr
```

## Worked example — värdeminskning

En mobiltelefon kostar **8 000 kr** ny. Den tappar **20 % per år** i värde. Skriv en funktion för värdet.

```
V(t) = 8 000 · 0,80^t
```

Värde efter 3 år:

```
V(3) = 8 000 · 0,80³
     = 8 000 · 0,512
     = 4 096 kr
```

Mobilen är värd lite mindre än hälften efter 3 år.

## Worked example — bakterietillväxt

Bakterier fördubblas varje timme. Du börjar med 100 bakterier.

```
B(t) = 100 · 2^t
```

(Här är a = 2 — fördubbling = 100 % ökning = förändringsfaktor 2.)

Efter 6 timmar:

```
B(6) = 100 · 2^6 = 100 · 64 = 6 400 bakterier
```

På 6 timmar har antalet ökat från 100 till över 6 000. Det är **exponentiell tillväxt** — den är otroligt snabb.

## Hur ser grafen ut?

**Tillväxt** (a > 1):
- Börjar lågt, men växer snabbare och snabbare
- Aldrig negativ (om C > 0)
- Kommer aldrig korsa x-axeln

**Avtagande** (0 < a < 1):
- Börjar högt, sjunker mot 0
- Aldrig negativ
- Kommer aldrig nå 0 — bara komma närmare och närmare

## Vanliga felaktigheter

**Misstag 1:** Att tro att exponentialfunktioner växer linjärt.
> "Om något ökar med 5 % per år blir det 10 % på två år"
> Fel! Det blir 1,05² = 1,1025, alltså **10,25 %**.

**Misstag 2:** Att räkna fel exponent.
> a^t betyder a multiplicerad med sig själv t gånger.
> a^0 = 1 (allt upphöjt till 0 är 1).
> a^1 = a.

**Misstag 3:** Att blanda ihop tillväxt och avtagande.
> Tillväxt: faktor STÖRRE än 1. (1,05, 1,12, 2)
> Avtagande: faktor MINDRE än 1 (men positiv). (0,90, 0,75, 0,5)

## Tillämpningar

Exponentialfunktioner finns överallt:

- **Ränta-på-ränta** — pengar växer
- **Värdeminskning** — bilar, mobiler, datorer tappar värde
- **Befolkningstillväxt** — om förhållanden tillåter
- **Bakterietillväxt** — fördubblas regelbundet
- **Halveringstid** — radioaktiva ämnen, läkemedel
- **Smittspridning** (under utbrott — innan immunitet bromsar)

I nästa lektion (3.8) lär vi oss en annan funktionstyp: **potensfunktioner**. De ser lika ut vid första anblicken, men är fundamentalt annorlunda.
